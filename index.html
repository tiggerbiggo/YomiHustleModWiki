Your Only Move is Hustle Modding Wiki


Welcome to the wiki! Currently it is very bare bones. You can help expand the wiki by making a pull request on github


This wiki will NOT cover how to decompile the game. Guides for this can be found in the on the YOMI Hustle Modding discord


Content roadmap

I myself am still getting to grips with how the game works but i'm fairly familiar with Godot's editor and GDScript, so i'm slowly making progress in my own understanding.

Currently I plan on documenting the basics for making new characters, as well as some weird consequences of certain actions in the godot editor.

As I make pages on each of these topics, I will add hyperlinks to the list. Once there are more topics, i'll start making pages to organise them and pull everything together in a more wiki style manner.
  • Creating a new character
  • Editing sprites and animations
  • Understanding the structure of the player scene
  • States and state variables
More will be added to this list as I learn more about the game, and if other people contribute to the wiki.
